# Голосование дома — как запустить (10 минут)

Файлы:
- `index.html` — страница голосования (её видят жильцы)
- `admin.html` — админка с паролем (полные данные + выгрузка PDF)
- `Code.gs` — «движок» для Google Таблицы (хранит голоса)

Сайт лежит на GitHub Pages, а голоса хранятся в твоей Google Таблице.

---

## Шаг 1. Создать Google Таблицу и «движок»
1. Зайди на https://sheets.google.com → создай пустую таблицу (назови, напр. «Голосование дома»).
2. В меню: **Расширения → Apps Script**.
3. Удали то, что там есть, и вставь весь код из файла **`Code.gs`**.
4. В начале кода поменяй пароль:
   `var ADMIN_KEY = 'СМЕНИ_ЭТОТ_ПАРОЛЬ_2026';` → напиши свой секретный пароль.
5. Нажми **Сохранить** (значок дискеты).
6. Справа вверху: **Deploy → New deployment**.
   - Нажми шестерёнку → выбери тип **Web app**.
   - **Execute as:** Me (твой аккаунт).
   - **Who has access:** **Anyone**.
   - Нажми **Deploy**, разреши доступ (Google спросит подтверждение).
7. Скопируй **Web app URL** (длинная ссылка вида `https://script.google.com/macros/s/..../exec`).

## Шаг 2. Вставить ссылку в сайт
1. Открой `index.html` любым редактором (Блокнот тоже подойдёт).
   Найди строку `var SCRIPT_URL = 'ВСТАВЬ_СЮДА_URL_APPS_SCRIPT';`
   и вставь между кавычками свою ссылку из шага 1.7.
2. То же самое сделай в `admin.html` (там такая же строка).
3. Сохрани оба файла.

## Шаг 3. Выложить на GitHub Pages
1. Зайди на https://github.com → **New repository** (напр. `golosovanie`), Public.
2. Кнопка **Add file → Upload files** → перетащи `index.html` и `admin.html` → Commit.
3. **Settings → Pages →** Source: ветка `main`, папка `/root` → Save.
4. Через 1–2 минуты появится адрес вида `https://твойлогин.github.io/golosovanie/`.
   - Голосование: `.../index.html`
   - Админка: `.../admin.html`
5. Эту ссылку на голосование рассылай жильцам.

---

## Что собирается с голосующего
- Номер квартиры + ФИО владельца + ответ (Так / Ні / Утримався).
- Устройство (iPhone/Android/ПК…), User-Agent, IP, город/регион/страна по IP.
- Точные GPS-координаты — если человек разрешит браузеру (если откажет — останется только город по IP).

## Правила
- Одна квартира = один голос. Повторно проголосовать нельзя, изменить голос нельзя.
- После голосования человек сразу видит результат: 3 столбика, высота = процент
  (Так — зелёный, Ні — красный, Утримався — жёлтый).

## Админка
- Открой `admin.html`, введи пароль (ADMIN_KEY из Code.gs).
- Видно статистику и таблицу всех голосов со всеми данными, GPS — ссылкой на карту.
- Две кнопки выгрузки PDF: **«кратко»** (только итог) и **«детально»** (каждый голос).

---

## Частые вопросы
- **Поменял Code.gs — изменения не видны.** Нужно заново задеплоить версию:
  Deploy → Manage deployments → ✏️ Edit → Version: **New version** → Deploy.
- **GPS не приходит.** Это нормально, если жилец нажал «Запретить». Город по IP всё равно сохранится.
- **Хочу очистить тестовые голоса.** Просто удали строки в Google Таблице (кроме шапки).
- **Точность по IP — это город, а не дом.** Так и должно быть; точный адрес даёт только GPS с разрешения.
